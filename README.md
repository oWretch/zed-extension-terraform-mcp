# Zed Extension for Hashicorp Terraform MCP Server

This extension provides access to the Hashicorp Terraform Model Context Protocol
(MCP) Server within the Zed editor. For more information, visit the
[Terraform MCP documentation](https://github.com/hashicorp/terraform-mcp-server).
